# 🚀 Scalable Backend API using FastAPI & MySQL

A production-ready backend system built using FastAPI, demonstrating REST API development, authentication, and database management with a clean and scalable architecture.

---

## 📌 Features

- 🔐 User Authentication (Signup/Login)
- 🔑 Password Hashing using bcrypt
- 🗄️ MySQL Database Integration
- ⚙️ SQLAlchemy ORM for database operations
- 🔄 Alembic for database migrations
- 📦 Modular and scalable project structure
- 📄 Data validation using Pydantic schemas

---

## 🛠️ Tech Stack

- **Backend:** FastAPI  
- **Database:** MySQL  
- **ORM:** SQLAlchemy  
- **Migrations:** Alembic  
- **Validation:** Pydantic  
- **Language:** Python  

---

## 📁 Project Structure
FASTAPI-SQL_Project/

│── models.py # Database models
│── schemas.py # Pydantic schemas
│── database.py # Database connection setup
